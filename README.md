## 🐳 Docker Compose Deployment

### Prerequisites
- Ubuntu 20.04+
- Docker Compose
- 4GB RAM available
- Git installed

### Deployment Steps

1. ✅  Clone Repository:
   ```bash
   git clone https://github.com/gndhmwn/final-project-digitalskola.git

2. ✅ Enter Project Directory:
   ```bash
   cd final-project-digitalskola
   
3. ✅ Start Containers:
   ```bash
   docker compose up -d
